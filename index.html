Lab #2

Lab#2:

I set up initial CSS styles, added a little JS to help the nav be more responsive, and created content for the home, about and contact us page.

I used jQuery to enhance the user experience by adding a hover effect to the images in the .page-link elements. When users hover over an image, its opacity decreases, providing visual feedback. This effect improves interactivity by clearly indicating which images are clickable. I achieved this using jQuery selectors to target .page-link and img elements, applying the opacity change on hover. This subtle visual enhancement makes the page feel more dynamic and encourages user engagement.

Lab #3

Lab#3

I finished the CSS page and added the services & giving pages.

I decided to enhanced the services page by adding two features: a fade-in animation and a scroll-triggered interactive component. The fade-in effect gradually reveals the text in the

elements inside each .gallery-item as they enter the viewport while scrolling. This creates a smoother, more engaging user experience by ensuring the content appears dynamically instead of all at once. The scroll-triggered interaction allows users to control when the content becomes visible, making the page feel more responsive and interactive. Overall, these enhancements improve the visual appeal and user engagement, offering a more polished and dynamic experience.

Lab #4

Lab#4

The interactive component I created for the "Ways to Give" section features clickable images and hover effects to enhance user engagement. Clicking an image triggers a fade transition, showcasing different donation options, while hovering reduces the image's opacity for subtle visual feedback. I chose this component to make the page more engaging and visually dynamic, encouraging users to interact with the donation options. The smooth animations align with the site's clean aesthetic, drawing attention to calls to action and improving the overall user experience.

Lab#5

The interactive component I chose for lab #5 is the accordion and I chose to use it on the services page. I implemented this to improve content usability and organization. The accordion allows users to easily navigate through different services by expanding or collapsing sections as needed, keeping the page clean and focused. This dynamic interaction reduces visual clutter, making it easier for users to find the information they need without being overwhelmed by long paragraphs of text.
